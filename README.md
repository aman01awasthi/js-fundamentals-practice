# 60-Day Fullstack Developer Roadmap
## Phase 1: JavaScript Foundations

Phase 1 of a 60-day journey to become a Fullstack + AI Developer. This repository contains JavaScript fundamentals practice covering core concepts through advanced async patterns.

## �📚 Course Structure

### Day 1: Functions & Function Scope
- Writing functions from scratch
- Understanding function declarations
- Scope and variable accessibility
- **Exercises**: greet, add, multiply, isEven, factorial functions

### Day 2: Arrays & Array Methods
- Array manipulation methods
- Working with collections
- Array searching and sorting
- **Exercises**: unshift, shift, pop, push, find, sort operations

### Day 3: Objects & Destructuring
- Object creation and property access
- Dot notation vs bracket notation
- Destructuring assignments
- Function destructuring patterns
- **Exercises**: Creating objects, adding/deleting properties, destructuring in functions

### Day 4: Control Flow & Loops
- Loop patterns and iteration
- Conditional logic
- Classic problem-solving (FizzBuzz)
- **Exercises**: For loops, while loops, nested iterations

### Day 5: Closures & Data Encapsulation
- Understanding closures
- Creating private variables
- Function factories
- **Exercises**: Counter with closure, data encapsulation patterns

### Day 6: Callbacks & Callback Patterns
- Function callbacks
- Higher-order functions
- Callback hell and its challenges
- **Exercises**: setTimeout callbacks, multi-step async operations

### Day 7: Promises & Async Flows
- Promise creation and chaining
- .then() chains
- Error handling with .catch()
- **Exercises**: Promise-based async operations, promise chaining

### Day 8: Async/Await & Advanced Patterns
- async/await syntax
- Cleaner async code without .then()
- Error handling with try/catch
- **Exercises**: Converting promises to async/await, complex async flows

## 🚀 How to Use

### Running Exercises
Each day has JavaScript files that can be run directly:

```bash
# Example: Run Day 1 functions
node Day1/functionsAndScope.js

# Example: Run Day 5 closures
node Day5/closures.js
```

### Viewing in Browser
Each exercise includes an HTML file for browser-based testing:

```bash
# Open any HTML file in your browser
open Day1/main.html
```
